About 1Screen

1Screen is a project created by me (Matt). The goal of 1Screen is to create a useful tool for creating and running tabletop RPG campaigns. It is also worth noting that this is an ever-evolving project so new additions and features will show up as time goes on.


If you ever need to contact me reach out via twitter @1ScreenApp. For those interested you can find our Terms & Privacy here.
